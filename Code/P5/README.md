This directory contains the code for the MF-SHRED. The solvers have been updated from OFELIA repository within the ERMETE Lab.
To generate directly the data, it is required to use dolfinx (version 0.10.0). The easiest way to do this is through conda:

```bash
conda create -n mfshred python=3.10
conda activate mfshred
python -m pip install gmsh
conda install -c conda-forge fenics-dolfinx=0.10 mpich pyvista ipykernel scipy
```

Then, the data can be generated by running the notebook `01_generate_snaps.ipynb`. This will generate the diffusion data and store it in the `data` directory.

Alternatively, the npz can be downloaded from Zenodo.
